**Qi 编程语言**（中文语法，文件后缀 `.qi`）整理的一份 **完整关键字表**，参考了 Rust 与 C 的语义设计，但表达方式采用中文，便于直观理解与书写。

---

## 🪄 **Qi 语言关键字总表**

### 📦 1. 程序结构与模块

| 关键字 | 含义              | 类似于（Rust/C）      |
| ------ | ----------------- | --------------------- |
| `包`   | 声明包名 / 模块名 | `mod` / `namespace`   |
| `导入` | 导入模块或库      | `use` / `#include`    |
| `公开` | 对外可见          | `pub` / `extern`      |
| `私有` | 模块内可见        | 无（Rust 中默认私有） |
| `作为` | 引入别名          | `as`                  |
| `常量` | 定义常量          | `const`               |
| `静态` | 静态变量          | `static`              |

---

### 🧠 2. 类型与变量

| 关键字     | 含义             | 类似于                        |
| ---------- | ---------------- | ----------------------------- |
| `变量`     | 声明变量（可变） | `let mut`（Rust） / `auto`    |
| `不可变`   | 声明不可变变量   | `let`（Rust）                 |
| `类型`     | 定义自定义类型   | `type` / `typedef`            |
| `结构体`   | 定义结构体       | `struct`                      |
| `枚举`     | 定义枚举         | `enum`                        |
| `联合体`   | 定义联合体       | `union`                       |
| `实现`     | 实现方法或 trait | `impl`（Rust）                |
| `特性`     | trait / 接口     | `trait`（Rust） / `interface` |
| `自我`     | 当前类型（self） | `Self`                        |
| `自身`     | 当前对象（this） | `self` / `this`               |
| `指针`     | 指针类型         | `*`（C）                      |
| `引用`     | 引用类型         | `&`（Rust）                   |
| `可变引用` | 可变引用         | `&mut`（Rust）                |

---

### 🧭 3. 控制流程

| 关键字 | 含义                | 类似于              |
| ------ | ------------------- | ------------------- |
| `如果` | 条件判断            | `if`                |
| `否则` | 否则分支            | `else`              |
| `匹配` | 模式匹配            | `match`（Rust）     |
| `循环` | 无限循环            | `loop` / `while(1)` |
| `当`   | 当条件为真时循环    | `while`             |
| `对于` | for 循环    统一        | `for`               |
| `中断` | 跳出循环            | `break`             |
| `继续` | 跳过当前循环        | `continue`          |
| `返回` | 返回函数结果        | `return`            |
| `跳转` | goto 跳转（极少用） | `goto`              |
